# 3D Models

Parametric 3D Models are stored in Enclosure/Parametric folder, containing Fusion 360 Files. 
For those, who prefer not to use Fusion360, parametric models are modifiable in [Makerworld](https://makerworld.com/en/models/2870548-touchord-open-source-midi-controller-shell)

For 3D Printing and custom tolerances Models can be downloaded in [Makerworld](https://makerworld.com/en/models/2870548-touchord-open-source-midi-controller-shell)

STEP Files are in Enclosure folder.

When 3D Printing, adjust tolerances and screw hole sizes accordingly. Recommended orientations:
- Base, flat side on the build plate;
- Top shell, top flat part on the build plate, tree supports (Requires extra removal);
- Keycaps, flat side on the bed;
- Key plates, flat on the bed.
